Flex Direction

If the flex-direction is set to row the main axis is horizontal, if it is set to column, the main axis is vertical:

  • row - main axis is horizontal (default)
  • column - main axis is vertical
  • justify-content - aligns items on the main axis

Dependent upon the flex-direction, the axis "opposite" the main axis is called the cross-axis. Since the default main axis is row (horizontal) the default cross-axis is column (vertical).

Using justify-content...

...the content can be spaced along the main axis:

  • flex-start - (default)
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

Using align-items...

...the content can be spaced along the cross-axis:

  • flex-start - (default)
  • flex-end
  • center
  • baseline - aligns the baselines of each items text
  • space-between
  • space-around
  • space-evenly

flex-wrap: wrap:

If you have specified "wrap" then you can use align-content. This will align the rows (or columns) within the flex container when there is extra space in the cross-axis. The values are the same as for justify-content.

You can then also use gap properties to set spacing between the rows and columns.
